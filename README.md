# SIMPLE-EXPENSE-TRACKER
## Overview
A basic Python program to track daily expenses. Perfect for beginners learning Python.

## Features
- Add new expenses with categories
- View all expenses
- See spending summary by category
- Delete expenses
- Automatic data saving

## How to Use
1. Save the expense_tracker.py file
2. Run: python expense_tracker.py
3. Follow the menu options

## Requirements
- Python 3.x
- No additional libraries needed
