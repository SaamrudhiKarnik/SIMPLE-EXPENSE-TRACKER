# Project Statement: Simple Expense Tracker

## Problem Statement
Many people find it difficult to track where their money goes each day.

## Project Scope
This program helps users record and view their daily expenses in a simple way.

## Target Users
- Python beginners
- People wanting to track basic expenses
- Students learning programming

## Features
1. Add expense records
2. View expense history
3. Categorize spending
4. Basic data management
